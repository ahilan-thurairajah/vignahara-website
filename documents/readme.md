# How To Guide

1. To start the website locally:
    a. In the "Terminal" window: "cd client"
    b. run: "npm run dev"
    c. The website should launch in [LOCAL URL](http://localhost:5173/)

2. Build Your Frontend Assets, run this command in "Teminal" window in "client" directory
    npm run build

3. To publish the website to Firebase, in the root folder run:
    a. firebase use vignahara-website
    b. firebase deploy --only hosting

4. Brand name font: 'Century Gothic', system-ui, 'Segoe UI', Roboto, sans-serif;
   Brand name font weight: 400

5. Slideshow background image size:
    Landscape:  1920px × 1080px
    Portrait:   1080px × 1350px
